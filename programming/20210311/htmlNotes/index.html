前端学习笔记之HTML5

本文最后更新于：19 小时前

¶Web标准

  • 结构：用于对网页元素进行整理和分类，现阶段主要是HTML
  • 表现：用于设置网页的外观样式，主要指CSS
  • 行为：指网页模型的定义及交互，现阶段主要是JavaScript

Web标准提出的最佳体验方案： 结构、样式、行为相分离。

¶基本语法概述

  1. HTML标签由尖括号包裹
  2. HTML标签通常成对出现，标签对中的第一个标签为开始标签，第二个带斜杠的为结束标签，例如<html></html>
  3. 有些标签为单标签，如：<br/>
  4. 标签之间有嵌套关系也有并列关系

¶HTML页面结构

¶基本结构标签

文档类型声明标签

<!DOCTYPE html>放在整个HTML文档的最前面，本质上不属于HTML标签，用来告诉浏览器当前文档的类型，示例中代表文档类型为HTML5

根标签

<html></html>页面中最大的标签，标签里放整个HTML文档的内容

其中开始标签中经常会加上指定文档语言的说明，如：<html lang="zh-CN">，实际上就算指定为英文网页也是可以显示中文的，这个属性一般会对搜索引擎有影响

头部标签

<head></head>HTML文档的头部，内容一般不显示在网页上

字符集声明标签

<meta charset="UTF-8">声明当前HTML文档存储的字符集，放在head标签中，重要

标题标签

<title></title>HTML文档的标题，放在头部标签中

主体标签

<body></body>HTML文档的主体，页面内容基本都是放在body标签里

¶块元素

块元素包含整个大区域的内容

标题标签（重要）

<h1></h1>标题共有6级（h1--h6），重要性依次递减，加了标题标签的文字会加大加粗，且在页面显示效果中一个标题独占一行

段落标签

<p></p>段落标签中的文字就是一个段落，段与段之间会有空隙，在没有手动换行的情况下，段落中的文字会根据浏览器的大小自动换行

水平线

<hr />水平线标签

HTML注释

<!-注释内容-->注释内容不显示在网页上

无序列表

无序列表与无序列表可以嵌套，每个元素前面是一个小点（无序）

有序列表

有序列表前面是元素的序号

表格

table：表格，caption：表标题，tr：行，td：单元格，th：列标题

table的属性：

属性 值 说明
border 数值（单位：像素） 规定表格边框的宽度
cellpadding 数值（单位：像素）或百分比 规定单元边沿与其内容之间的空白
cellspacing 数值（单位：像素）或百分比 规定单元格之间的空白
width 数值（单位：像素）或百分比 规定表格的宽度

th的属性（可选）

属性 值 说明
colspan 数值 跨列居中（合并居中）
rowspan 数值 跨行居中（合并居中）

引用（块引用）

块引用内格式与引用外独立

预格式化文本

<pre></pre>将块内的内容按照原格式原样输出

音视频

<audio>提示内容</audio>播放音频，提示内容为浏览器无法播放时的提示信息

播放格式：mp3、wav等

<video>提示内容</video>播放视频，提示内容为浏览器无法播放时的提示内容

播放格式：mp4、ogg、webm、mkv等

div和span分区标签

<div></div>一个div标签独占一行，即横向分区

<span></span>span标签为纵向分区

¶行内元素

行内元素影响少量的内容，必须嵌套在块元素中

图像标签

<img src="图像URL" />单标签

图像标签的其他属性：

属性 属性值 说明
src 图片路径 略（必填）
alt 文本 当图片不能显示时的替换文字
title 文本 将光标放到图片上时的提示文字
width 数值（单位：像素） 设置图片的宽度
height 数值（单位：像素） 设置图片的高度
border 数值（单位：像素） 设置图片的边框粗细

链接标签

<a href="URL">文本</a>双标签，标签中间写显示在页面上的可以点击的文本，标签内的href属性中为目标URL，可以为绝对或相对URL

强制换行

<br />换行标签，在标签处换行

文字格式化标签

<strong></strong> 或 <b></b>文字加粗

<em></em> 或 <i></i>文字倾斜

<del></del> 或 <s></s>删除线

<ins></ins> 或 <u></u>下划线

引用（行内引用）

<q></q>在行内使用，被引用的内容会被加上引号

字符实体

在HTML中显示一些特殊符号时需要用字符实体，如<>&和多个空格，有两种方式：

  • 用实体名：&实体名
  • 用实体数字：&#实体数

具体实体对应的字符此处略

预格式化文本--计算机代码

<code></code>将文本在类似于代码框中显示
